# 地方恒星時を単独で計算するロジックを作ったブランチ

## 2022.03 時点では必要なくなった機能だけど、バックアップ的に残しておきます

環境構築

```
git clone git@github.com:SonicGarden/yorozu-uranai.git
cd yorozu-uranai
yarn install
```

テスト

```
yarn test
```

Next.js 開発サーバー起動

```
yarn dev
```

Next.js 本番ビルド・本番サーバー起動

```
yarn build
yarn start
```
